CONFIGURA GOOGLE SHEETS

Para guardar en la nube necesitas conectar tu Google Sheet.
Pasos:
1. Abre Google Sheets y crea una hoja nueva.
2. Ve a Extensiones → Apps Script.
3. Pega el código del script (ver instrucciones abajo).
4. Despliega como Web App → cualquiera puede acceder.
5. Copia la URL del despliegue y pégala abajo.

Esta URL solo se aplica en este navegador. Para que funcione en cualquier dispositivo sin configurar nada, pégala en DEFAULT_SHEET_URL dentro del código y sube el archivo de nuevo.
